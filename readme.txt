FreeCharge is a gateway plugin that extends WooCommerce, allowing you to take payments via Freecharge. 

== Description ==
Freecharge cooperates with Visa and Mastercard Credit Cards and Debit cards and Freecharge Wallet. This means that you easily can take freecharge wallet payments as well as direct payments via any of the associated banks.

When the order goes through, the user is taken to Freecharge to make a secure payment. No SSL certificate is required on your site. After payment the order is confirmed and the user is taken to the thankyou page.

Github : https://github.com/vikas5914/woo-freecharge

== IMPORTANT NOTE ==

In order to use the Freecharge gateway you need a Freecharge merchant account. 
Please email at merchantsupport@freecharge.com for merchant account .

== Installation ==
1. Ensure you have latest version of WooCommerce plugin installed (WooCommerce 2.1+)
2. Unzip and upload contents of the plugin to your /wp-content/plugins/ directory
3. Activate the plugin through the Plugins menu in WordPress
